## Common Parameter Patterns

### Query Methods
```bash
midclt call <namespace>.query '[[["field", "=", "value"]]]' '{"extra": {"properties": ["prop1"]}}'
```

### Create Methods
```bash
midclt call <namespace>.create '{"field1": "value1", "field2": "value2"}'
```

### Update Methods
```bash
midclt call <namespace>.update <id> '{"field": "new_value"}'
```

### Delete Methods
```bash

```
